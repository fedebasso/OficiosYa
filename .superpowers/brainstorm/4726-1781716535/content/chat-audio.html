¿Cómo se graban los audios?

Hay dos formas de enviar audios — ¿cuál es más natural para el usuario?

A

Mantener presionado para grabar (como WhatsApp)

El usuario mantiene apretado el botón 🎙️ mientras habla. Al soltar, se envía el audio. Experiencia nativa mobile, muy familiar.

🎙️ Mantené presionado para grabar...
B

Tocar para iniciar / tocar para detener

Un toque inicia la grabación, otro la detiene y muestra preview antes de enviar. Más control, menos presión sostenida.
